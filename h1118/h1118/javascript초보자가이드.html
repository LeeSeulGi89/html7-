    📚 학습 가이드

    1. 기본 개념

    • 변수 (let, const): 데이터를 저장하는 공간
    • 함수 (function): 반복되는 코드를 묶어놓은 것
    • 조건문 (if-else): 조건에 따라 다르게 동작
    • 반복문 (for, while): 같은 작업을 여러 번 반복
    • 배열 (Array): 여러 데이터를 순서대로 저장

    2. DOM 조작

    • getElementById(): id로 요소 찾기
    • querySelector(): CSS 선택자로 요소 찾기
    • innerHTML: HTML 내용 변경
    • innerText: 텍스트만 변경
    • style: CSS 스타일 변경

    3. 이벤트

    • onclick: 클릭했을 때
    • onmouseover: 마우스를 올렸을 때
    • onload: 페이지 로딩 완료 시
    • onchange: 값이 변경되었을 때

    4. 학습 순서 추천
